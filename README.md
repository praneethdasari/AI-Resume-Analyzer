# AI Resume Analyzer

AI Resume Analyzer is a web application that analyzes resumes using Generative AI and provides suggestions to improve them.

The application reads the resume content, evaluates it using Google Gemini AI, and generates useful insights such as job match score, missing skills, and improvement tips.

---

## Features

- Upload resume in PDF format
- AI-powered resume analysis
- Job match score estimation
- Missing skills detection
- Resume improvement suggestions

---

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- PDF processing using pdfplumber

---

## Installation

Clone the repository

git clone https://github.com/praneethdasari/AI-Resume-Analyzer.git

Go to the project directory

cd ai-resume-analyzer

Install required dependencies

pip install -r requirements.txt

---

## Run the Application

Run the Streamlit app

streamlit run app.py

Then open your browser and go to

http://localhost:8501
